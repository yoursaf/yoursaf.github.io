TMUX-terminal multiplexer

发表于 2017-10-26   |   分类： Linux

TMUX is a terminal multiplexer: it enables a number of terminals to be created,accessed, and controlled from a single screen. tmux may be detached from a screen and continue running in the background,then later reattached.

1. 安装

Archlinux

sudo pacman -S tmux

Ubuntu

sudo apt-get install tmux

2. 启动

$ tmux new -s yoursession

3. 基本设置

tmux的所有快捷键操作都需要前缀支持，默认前缀为C-b。因与Emacs冲突，更改为C-z

unbind ^b
set -g prefix 'C-z'

4. 常用快捷键

1. 窗口快捷键

  • c 创建新窗口
  • n 选择下一个窗口
  • p 选择前一个窗口
  • | 最近一次活跃窗口之间进行切换
  • 0~9 按编号选择窗口
  • , 重命名窗口
  • . 更改窗口编号，但只能更改为未使用的窗口编号
  • & 关闭窗口
  • w 以菜单方式选择窗口
  • f 在所有窗口中查找内容

    2. 面板快捷键

  • “ 横向分割面板
  • % 纵向分割面板
  • o 跳到下一个面板
  • x 关闭面板
  • ; 切到最后一个使用的面板
  • 方向键 面板之间切换
  • 空格键 切换面板布局

5. 常用命令

1. tmux ls 在终端下列出tmux会话

2. 新建会话

  • tmux 直接新建会话并进入
  • tmux new -s sessionname 创建一个指定名称的会话
