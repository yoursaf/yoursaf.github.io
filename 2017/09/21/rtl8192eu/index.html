Archlinux rtl8192eu无线网卡驱动安装

发表于 2017-09-21   |   分类： Linux ， Archlinux

rtl8192eu无线网卡驱动下载地址：

masterzorag’s github
Mange’s github


1. 安装

dkms

sudo pacman -S dkms
sudo dkms add .
sudo dkms install rlt8192eu/1.0

make

sudo make && sudo make install

2. 加载模块

sudo modprobe 8192eu

3. 禁用内核自带模块

sudo touch /etc/modprobe.d/rtl8xxxu.conf

在rtl8xxxu.conf中加入如下语句：

blacklist rtl8xxxu
